# Fullstack 16-Layer Architecture

This repository is a learning and reference project for building a production grade full stack system using React, Go, Postgres, Docker, and Github.

The goal is to understand and implement a 16-layer application architecture used in real-world systems (THESIS - Evaluation of the Technological Efficacy and Decision Supoort Utitlity of an RFID-Enabled School Meal System within an Institutional Cafeteria).
